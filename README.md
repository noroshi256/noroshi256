## O que estou construindo

**`NoroshiOS`** — OS bare-metal em Assembly x86 + C  
Bootloader, GDT, modo protegido, framebuffer VGA, kernel em C.  
Sem SO embaixo. Sem libc. Roda no QEMU.  
*Próximo: IDT, interrupções de teclado.*

**`Jornada IA`** — do zero até LLMs  
Começando pela matemática (álgebra linear, cálculo, estatística).  
Cada fase com exercícios implementados à mão antes de usar biblioteca.  
*Fase 1 em andamento.*

**`Jornada Eletrônica`** — eletrônica analógica  
Sete fases: componentes passivos → digital → Arduino → ESP32 → ROS2 → IA embarcada → segurança IoT.  
*Fase 1 pendente — kit montando.*

**`Noroshi Tools`** — kit bash para uso em campo  
12 scripts modulares: rede, SSH, backup, inventário, logs, RDP, VPN, checklists.  
Construído para uso real, não para portfólio.
